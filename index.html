Who am I?

I am a former mechanical engineer turned full-stack developer. I have a diverse background that includes experience in aerospace, automotive manufacturing, supporting the International Space Station program, wine production, and viticulture. I helped create and maintain flight planning tools for NASA while working at Barrios Technology. That's when I discovered that I enjoyed programming and felt like I may have missed my calling as a developer. The seed was planted and a year later I decided to make a career change and pursue web development full time.

Meet Me Halfway

Meet Me Halfway helps users find a meeting point halfway between them, measured by driving time. It then displays restaurants near that meeting point. I worked on using the Google Maps API to create a map, funnel data into an algorithm I wrote to find the halfway point, and connecting the Zomato API to our map to display the restaraunts.

Nabi Freedom

Nabi Freedom is a local non-profit here in Houston that assists non-violent prisoners. I worked on setting up tables in our database using Sequelize, user registration, login, and authentication, encrypting user passwords using bcrypt, and creating a user-friendly administration page so non-technical users can maintain and edit the site.

Theros Beyond Death

Theros Beyond Death is a website dedicated to the newest set released in early 2020 of the popular Magic the Gathering card game. Users can look at and filter cards to their heart's content. The site is built on Node.js and is a full-stack project where I did all of the front-end work as well.
