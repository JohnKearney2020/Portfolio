Northstar Cafe Website

This project is a static website built using HTML, CSS, and Bootstrap. It is a single page application meant to give visitors all the information they need about Northstar Cafe in one easy to navigate page full of eye catching animations and smooth transitions. Users can find locations and menus, see pictures of some of the popular dishes, see real user reviews from Yelp, and use a contact form to contact the business. All of that is accomplished while being fully mobile responsive.
